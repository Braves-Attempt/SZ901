# SZ901高速FPGA下载器简介

SZ901是一款专业的高速FPGA网络下载器，集高性能、多功能和灵活性于一体，专为AMD(Xilinx) FPGAs设计。

## 主要特点
- 仅支持Vivado，不支持ISE
- 专为FPGA高速下载设计，最高支持53M。
- 支持4路JTAG同时工作，支持同时仿真,烧写
- 网络接口，使用灵活
- 专属上位机，烧写程序更加灵活，并支持国产Flash
- 支持烧写文件格式转换及coe波形文件自动生成


## 应用场景
SZ901适用于FPGA开发、调试和并发任务处理，广泛用于硬件开发、嵌入式系统设计等领域。其强大的约束支持和灵活的配置使其成为专业开发者的理想选择。
